# 赛题
提升用户体验的同时降低运营成本是云服务竞争力的关键。
在视频直播场景中，网络成本是影响服务成本的关键因素之一，不同的流量调度方案 会产生不同的网络使用成本。本赛题以华为云视频直播服务流量调度问题为基础，并 进行一定的抽象、调整和简化。参赛选手需要设计高效的调度算法，在满足客户要求 的前提下，通过对流量的合理调度，最小化网络使用成本。
